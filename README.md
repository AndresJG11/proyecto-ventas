## Usage

### Run in dev mode

```zsh
# development mode
$ yarn dev
```

### Build a production build

```zsh
$ yarn build
$ yarn build:linux
$ yarn build:win64
$ yarn build:win32
```

Check dist folder to get .AppImage for Linux and .exe for windows
